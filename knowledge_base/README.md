# База знаний курса

Поместите сюда материалы курса в следующих форматах:

- **PDF** (.pdf) - лекции, презентации
- **TXT** (.txt) - текстовые материалы
- **MD** (.md) - Markdown файлы
- **DOCX** (.docx) - документы Word

Бот автоматически загрузит все файлы при первом запуске и создаст векторную базу данных.

## Пример структуры:

```
knowledge_base/
├── module1_lecture.pdf
├── module2_presentation.pdf
├── faq.txt
├── course_description.md
└── ...
```

После добавления файлов перезапустите бота, чтобы он переиндексировал базу знаний.
